# Systematic Literature Review Validation

## Introduction

The vector data validation system has been expanded to enable data from systematic literature review (SLR) to be validated using the same tool and workflow.

## What's different?

#### Data submission workbook template

An Excel data submission template has been developed for each vector species group. This contains reference tables for all countries, allowing data to be submitted from anywhere in the same worksheet.

The required columns and validation rules have been updated to meet SLR requirements (see below).

#### Email address

The email address for submitting SLR data is: [slr.validation@efsa.epimundi.com](mailto:slr.validation@efsa.epimundi.com)

#### Required columns

The required and optional columns are slightly different from the standard data validation. Check the [columns](/validation/columns/) reference for details.

#### Validation rules

A new set of validation rules has been created, adapted to the SLR. In particular, these check for *bibliographicCitation* and *occurrenceRemarks*. See the [rules](/validation/rules/) documentation for more details.

## What hasn't changed?

The overall validation process remains the same.

- Submit data via email to the validation tool
- Check any errors and resubmit to ensure they are valid

If the tool is being used by reference entomologists undertaking SLRs, there is no need to submit for manual validation.

However, the tool may also be used by some entomologists for different reasons. In this case, the same manual validation submission option is available.
